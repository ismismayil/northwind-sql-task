# DevJoint Week 1: From SQL Basics to Advanced

This repository contains SQL solutions for the Week 1 tasks based on the Northwind relational database schema.

## Content Summary
- **Basic Queries:** Data selection, filtering (`WHERE`), sorting (`ORDER BY`), and pagination (`LIMIT`).
- **JOIN Operations:** Multi-table relationships using `INNER JOIN`, `LEFT JOIN`, and `SELF JOIN`.
- **Aggregations:** Grouping and metrics calculations with `GROUP BY`, `HAVING`, `SUM`, `AVG`, and `COUNT`.
- **Advanced Logic:** Multi-step analytics using Subqueries and Common Table Expressions (`WITH` clause).
- **Window Functions:** Ranking and analytical sequencing with `ROW_NUMBER()`, `RANK()`, and running totals using `SUM() OVER()`.
- **Query Optimization:** Index creation and converting correlated subqueries into set-based `JOIN` operations.
